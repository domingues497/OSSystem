# OSSystem

## Tecnologias

### Front-end (Dashboard/Kanban)
- HTML + CSS + JavaScript (templates do Flask)
- Chart.js (gráficos)
- Font Awesome (ícones)

### Back-end
- Python
- Flask (aplicação em [rtf_generator](file:///d:/trabalhos%20faculdaded/OSSystem/rtf_generator))
- Gunicorn (deploy em Linux)
- Nginx (proxy reverso no deploy)

### Integrações / Bibliotecas
- psycopg2 (conexão com banco PostgreSQL do ERP)
- python-dotenv (variáveis de ambiente via `.env`)
- requests (integração com Telegram API)

## Bancos de Dados
- PostgreSQL (ERP): conexão configurada via variáveis `ERP_DB_HOST`, `ERP_DB_PORT`, `ERP_DB_NAME`, `ERP_DB_USER`, `ERP_DB_PASS`
